Template Management

Templates are reusable html pages that provide an easy way to change the look and feel of a web page or application.

Templates contain the basic html structure for needed for rendering a page. The content page should only provide the html needed to render the content of the page.

Before you can use a template you need to identify the content area of the template by assigning the "master-content" css class name to an element withing the template.

<!DOCTYPE html>

<html>
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8"/>
        <title>My Website</title>
    </head>

    <body>
        <h2>My Website</h2>
        <div class="master-content">
            <!-- your content will be loaded here -->
        </div>
    </body>

</html>

To set or change the template of a web page,you will need to declare and set the $masterTemplate page property as shown below:

<?php

    require_once('raxan/pdi/autostart.php');

    class NewPage extends RaxanWebPage {

        protected $masterTemplate = 'mytemplate.html';

        protected function _load($e) {
            $this->content('Hello World');
        }
    }

?>

In addition to setting the $masterTemplate of the page, you can optionally set the $masterContentBlock property to use your own template content css selector. The default template content selector is ".master-content".
