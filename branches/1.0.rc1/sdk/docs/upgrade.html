Upgrading from a previous version

Before upgrading your application or web site you should first check the What's new and ChangeLog for new and breaking changes. Also remember to properly test applications with the new version before deploying to production servers.

To upgrade from a previous version, download and unzip the latest version, then copy overwrite the existing files with the latest version.

It you're having problems upgrading from a previous version then please see the Troubleshooting Guide
