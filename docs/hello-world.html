A simple Hello World App

Note: This tutorial assumes that you're already familiar with the basic php programming concepts. If you're new to php please visit http://www.php.net/getting-started to learn about the basics of the language.

To get started with PDI, we will explore what it takes to create a simple Hello World application.

To begin, simply create a file inside the root of your app folder called hello.php. For example: c:\website\app\hello.php

Next, we need to edit the page and include the raxan/pdi/gateway.php script.

<?php
include_once('raxan/pdi/gateway.php');
?>

The pdi/gateway.php file will load all the necessary classes and functions that we will need to continue with our tutorial.

Create an instance of the RichWebPage class.

$page = new RichWebPage();

Append the words "Hello World" to the body tag

$page['body']->append('Hello World');

Finally, we need to send the reply to the client.

$page->reply();

That's it! You can try running the script inside your browser to view the results. Here's the complete code:

<?php
include_once('raxan/pdi/gateway.php');

$page = new RichWebPage();
$page['body']->append('Hello World');
$page->reply();

?>

In addition to the above, we can use the globally accessible P() function to access the default Web Page Controller object.

<?php
    P('body')->append('Hello World');
    P()->page->reply();
?>
