Troubleshooting

Raxan for PHP not working

If you're having problems loading JavaScript and/or styles sheets or binding to an Ajax event, then you need to set the site.path and site.url options inside your config file. Please note that paths and urls must have a trailing slash (/).

To create a custom configuration file please see Loading a Custom Config file

Events are not working

Make sure the url path to the raxan folder is accessible from the web browser. You might also want to manually confiugre the raxan.path and raxan.url. See Loading a Custom Config file for more information
