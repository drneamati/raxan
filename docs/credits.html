Credits / Acknowledgments

The jQuery Library
jQuery is a light-weight but powerful cross-browser JavaScript library that is used to power the Raxan Framework. Visit http://jquery.com/. See also John Resig's wonderful tips, tweaks and suggestions at http://ejohn.org/. His website has been an inspiration on matters relating to Javascript development.

BluePrintCss
A Cross browser CSS framework. The Raxan CSS framework is based on BlueprintCSS. Visit http://code.google.com/p/blueprintcss/

BroilerplateCss
This is another CSS framework that is based on BluePrintCss. Visit http://code.google.com/p/css-boilerplate/

Drop Shadow by Phil Baines
Inspired the creation of the Shadow box wrappers. Visit http://www.wubbleyew.com/tests/dropshadows.htm

CSS to Xpath PHP converter
Many thanks to developer (php_penguin) who had ported Joe Hewitt's JavaScript CSS-XPath library to PHP.
