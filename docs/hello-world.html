A simple Hello World App

Note: This tutorial assumes that you're already familiar with the basic php programming concepts. If you're new to php please visit http://www.php.net/getting-started to learn about the basics of the language.

To get started with Raxan for PHP, we will explore what it takes to create a basic Hello World application.

To begin, we will create a php web page inside the root of our app folder called hello.php. For example: c:\website\app\hello.php

Next, we need to edit the page and include the raxan/pdi/autostart.php script.

<?php require_once('raxan/pdi/autostart.php'); ?>

The pdi/autostart.php file will load all the necessary classes and functions that we will need to continue with our tutorial. It will also automatically initialize our page class.

Create a new sub class of RaxanWebPage class called HelloWorldPage with a _load() function:

<?php
class HelloWorldPage extends RaxanWebPage {

    protected function _load() {

    }

}
?>

Select the page body tag element and append the words "Hello World" to it:

    protected function _load() {
        $this['body']->append('Hello World');
    }

That's it! You can try running the script inside your browser to view the results. Here's the complete code:

<?php
    require_once 'raxan/pdi/autostart.php';

    class HelloWorldPage extends RaxanWebPage {

        protected function _load() {
            $this['body']->append('Hello World');
        }

    }
?>

When the page loads the system will initialize HelloWorldPage class as the default web page controller.

In addition to the above, we can use the globally accessible P() function to access the default Web Page Controller object.

<?php
    P('body')->append('Hello World');
    P()->page->reply();
?>

To learn more about the P() function, see the Global Functions user guide.

To learn how to interact with elements on a page, see Connecting the Dots.
