Cross Browser Support

Built-in support for mobile enabled web pages.

Easy Pagination

Create a data Pager just a single method call.

CSS Framework

Predefined classes for quick and easy styling

Plugin Support

Extend the framework with custom plugins and extensions

Multilingual Support

Supports Internationalization and Localization

Object Oriented Design
