Software Requirements

Server Requirements

  • HTTP Web Server (e.g. IIS, Apache, etc).
  • PHP 5.1 or higher is required for PHP development. Version 5.2+ recommended

Browser Requirements

  • Supported Browsers: IE6+, FF 2+, Safari 3+, Chrome, Opera 9+
  • JavaScript must be enabled for Ajax operations
  • Browser Cookies must be enabled when using sessions

Mobile Requirements

  • WAP 2.0 enabled device
  • Smartphones with Full HTML Web browser (e.g. iPhone, Palm Pre, Android based phones, etc)
