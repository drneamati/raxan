Getting the Latest Source Code

The Raxan Source Code is available from Google's SVN repository.

To check out the source code you will need and SVN client such as TortoiseSVN or the SVN command line utility.

For example:

svn checkout http://raxan.googlecode.com/svn/trunk/raxan/ raxan-read-only

For other standalone clients please visit http://subversion.tigris.org/links.html#clients
