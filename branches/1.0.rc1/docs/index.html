User Guide for Web Application Developers

Welcome to the Raxan User Guide for Web Application Developers. The purpose of this user guide is to help developers understand how to build and develop applications using the Raxan Framework.

What is Raxan?

Raxan is an Open Source Rich Ajax Application Framework that combines the unique features of CSS, HTML, JavaScript and PHP to deliver an integrated solution for building today's rich web 2.0 applications.

What is Included?

For the Ajax/CSS developer, Raxan comes bundled with the latest jQuery libraries, a dynamic JavaScript/CSS loader and a CSS framework. The CSS framework includes support for grid and elastic layouts, columns, tables, and a lot of other useful features for creating simple to complex web pages with just a few lines of code.

For the PHP developer, Raxan comes bundled with the PDI (Programmable Document Interface) Framework, which includes support for client-server Ajax communications, server-side jQuery DOM manipulation API and a lot of other powerful features.

Main features

Cross Browser Support

Supports IE 6+, Opera 9+, FF 2+, Safari, and Chrome

Template Binder

Bind any PDO dataset or array to elements within a web page

Mobile Support

Built-in support for mobile enabled web pages.

Easy Pagination

Create a data Pager just a single method call.

CSS Framework

Predefined classes for quick and easy styling

Plugin Support

Extent the framework with custom plugins and extensions

Multilingual Support

Supports Internationalization and Localization

DOM Manipulation

Server-side HTML/XML DOM Traversing and Manipulation

Data Sanitization

Validate and sanitize user inputs

Object Oriented Design

Includes support for Chainable method calls

Integrated Ajax Support

Performs ajax file uploads and event callbacks

CSRF/XSS protection

Automatically blocks and prevents cross-site request forgery

Embedded Applications

Easily integrate Raxan Ajax applications into any web page.

Switchboard

Manage multiple page actions with the with switchboard.
