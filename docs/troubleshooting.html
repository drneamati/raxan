Troubleshooting

PDI not working

If you're having problems loading JavaScript and/or styles sheets or binding to an Ajax event, then you need to set the site.path and site.url options inside your config file. Please note that paths and URLs must have a trailing slash (/).

To create a custom configuration file please see Loding a Custom Config file
