Overview

The Raxan (Rich Ajax Application) Framework is a combination of Cascading Style Sheets and Javascript libraries that can be used to create accessible, standard compliant and elegant looking web sites and applications with enhanced user interface features and experience. The Framework is bundled with the latest jQuery libraries and a CSS framework that makes it very easy to create aesthetically beautiful web pages with a few simple and easy to use CSS classes.

Below is a list of features found in the framework:

Script Support
  • Automatic inclusion of code-behind JavaScript file
  • Dynamically include additional js and css files
  • Support for ready, load and unload events
  • Support for multiple (comma delimited) startup files
  • Built-in support for the jQuery library.
  • Can be customized to work with other Javascript libraries/frameworks.
CSS Framework
  • Predefined classes for quick and easy styling
  • Easily create complex layouts
  • Elastic cells and layouts
  • Reusable themes and templates
Plugin Support
  • Cursor and TabStrip plugins - Since version 0.2

Raxan's small footprint and dynamic css/javascript loading capabilities makes it ideal for just about any web application or website. You can choose the components and libraries that you want on your web page, and Raxan will automatically detect and load the library or style sheet even after the page has already been loaded.

With Raxan's CSS framework you can easily go from simple fixed-width designs to complex multi-column elastic layouts with just a few lines of html code. No Javascript is required to accomplish multi-column layouts, thus making it ease to create nice looking, accessible web pages in minutes. But if you want to spice up your web pages with interactive desktop-like functionalities, then you can easily accomplish this with the jQuery interactive APIs.

To learn more about the Raxan Framework visit the Getting Started with Raxan web page.
