Roadmap

The following list is by no means complete. If you would like to contribute to the development of the Raxan framework you're more than welcome to do so. We are always looking out for new ideas and solutions that can help to improve the quality and productivity of the framework. Things to do:

Version 1.0

  • Plugin Support
  • Code Optimization
  • Selector Optimization
  • Optimize DOM Manipulation API

Version 1.1

  • Full & Partial page caching
  • New Web Page Themes
  • More plugins and templates

Version 1.5

  • Improve Database support
  • Built-in ORM support

Future versions

  • Support for Nested Templates
  • Support for html.include('namespace/*')
