Credits/Acknowledgements

The jQuery Library - This is a very light-weight but powerful cross-browser JavaScript library that is used to power the Raxan Framework. Visit http://jquery.com/. See also John Resig's wonderful tips, tweaks and suggestions at http://ejohn.org/. His website has been an inspiration on matters relating to Javascript development.

BluePrintCss - Cross browser CSS framework. The Raxan CSS framework is based on BlueprintCSS. Many thanks to the BlueprintCSS team for the wonderful contributions they have made to the CSS world. Visit http://code.google.com/p/blueprintcss/ and http://www.blueprintcss.org/.

BroilerplateCss - This is another CSS framework that is based on BluePrintCss. Visit http://code.google.com/p/css-boilerplate/

Drop Shadow by Phil Baines - Inspired the creation of the Shadow box wrappers. Visit http://www.wubbleyew.com/tests/dropshadows.htm

A List Appart - Sliding Doors - Inspired the creation of the TabStrip plugin. Visit http://www.alistapart.com/articles/slidingdoors/
